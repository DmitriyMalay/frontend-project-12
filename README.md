**- leo-profanity**  
**- Rollbar**  

## Features

- User registration and authentication  
- Create, rename, and remove channels  
- Send and receive messages in real time  
- Profanity filtering  
- Error tracking and notifications

```sh

cd frontend-project-12
make install
make build
make start
```
